# java-starter
![Build](https://github.com/rbento/java-starter/actions/workflows/gradle.yml/badge.svg)

A simple Java project starter.

### Dependencies

- [Gradle](https://gradle.org/releases/) 8.4
- [JDK 21](https://www.azul.com/downloads/?package=jdk#zulu) from Azul

### Build

```bash
./gradlew clean build
```
